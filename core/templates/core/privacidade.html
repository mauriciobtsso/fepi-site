Política de Privacidade e Termos de Uso

Última atualização: {% now "d/m/Y" %}


1. Introdução

A Federação Espírita Piauiense (FEPI) compromete-se com a transparência e a proteção dos dados pessoais dos seus usuários. Esta política descreve como coletamos e utilizamos as informações no nosso portal.

2. Coleta de Dados

Coletamos informações mínimas necessárias para a interação com o site:

  • Formulário de Contato: Nome, e-mail e telefone fornecidos voluntariamente para resposta a dúvidas.
  • Logs de Acesso: O servidor pode registrar o IP de acesso para fins de segurança e estatística, sem identificar o usuário diretamente.

3. Uso das Informações

Os dados coletados são utilizados exclusivamente para:

  • Responder às mensagens enviadas pelo "Fale Conosco".
  • Melhorar a experiência de navegação no portal.
  • Fins administrativos internos da Federação.

Não compartilhamos seus dados com terceiros para fins comerciais ou de marketing.

4. Cookies

Este site utiliza cookies técnicos essenciais para o funcionamento do sistema (como manter a sessão ativa) e cookies de análise (Google Analytics) para entender como o site é utilizado, sempre de forma anônima.

5. Direitos do Titular (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD), você tem o direito de solicitar a visualização, correção ou exclusão dos seus dados pessoais armazenados por nós. Para isso, entre em contato através dos canais oficiais.
